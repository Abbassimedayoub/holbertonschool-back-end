# Back End — REST API Scripts (Python)

## Overview

Python scripts that consume a public REST API (JSONPlaceholder) to report an employee's TODO-list progress and export the data in several formats. A classic Holberton School exercise on API consumption and data serialization.

## Content

| Script | Role |
|---|---|
| `api/0-gather_data_from_an_API.py` | Fetch and display an employee's TODO progress by ID |
| `api/1-export_to_CSV.py` | Export the employee's tasks to CSV |
| `api/2-export_to_JSON.py` | Export the employee's tasks to JSON |
| `api/3-dictionary_of_list_of_dictionaries.py` | Export all employees' tasks to a single JSON file |

## Usage

```bash
python3 api/0-gather_data_from_an_API.py <employee_id>
```

## Technologies

Python 3, `requests`, `csv`, `json`.

## Skills Demonstrated

Consuming REST APIs, parsing JSON responses, and exporting structured data to CSV/JSON.
